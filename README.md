# Carnaval 2027 • Painel Operacional (sem senha)

## Antes de publicar
1. Abra `js/config.js` e preencha `supabaseUrl` e `supabaseAnonKey`.
2. No Supabase, execute `sql/supabase_access_policies.sql`.
3. Publique os arquivos no GitHub Pages.

## Importante
Neste modo sem senha, qualquer pessoa com acesso ao site poderá visualizar e editar o planejamento operacional.

## Arquivos principais
- `index.html`
- `css/styles.css`
- `js/app.js`
- `js/config.js`
- `sql/supabase_access_policies.sql`
- `sql/supabase_sem_senha_patch.sql`
